# v0.1.2

## Highlights
- Added an explicit provider selector in Unity Code Agent settings so Copilot and BYOK configuration are separated.
- Copilot sessions now ignore stale BYOK base URL and API key fields, reducing accidental provider misrouting after switching modes.

## Improvements
- BYOK setup now requires a full HTTPS BaseUrl before creating provider configuration.
- The settings inspector only shows BYOK BaseUrl and ApiKey fields when BYOK is selected.
- Added a solution file for opening and working on the service project from standard .NET tooling.

## Fixes
- Updated no-model-selected guidance to tell users to select a model and retry.

## Upgrade Notes
- No migration is required. Users who want a custom provider should select BYOK and enter a full HTTPS BaseUrl.
